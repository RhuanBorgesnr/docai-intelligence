## Versao Tecnica (Para TI/Desenvolvedores)

> DocAI Intelligence e uma plataforma de processamento documental com IA.
>
> Stack: Django + React + PostgreSQL + Celery/Redis.
>
> Usa Groq (Llama 3 70B) para extracao inteligente, com fallback para regex em documentos padronizados.
>
> API REST completa, busca semantica com embeddings, processamento assincrono.
>
> Pode rodar on-premise ou cloud. Integracao via API com ERPs.
